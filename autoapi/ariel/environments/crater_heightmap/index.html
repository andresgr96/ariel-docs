Spawn a robot into the crater world.

Parameters:

  • mj_spec (mujoco.MjSpec) – The MuJoCo specification for the robot.

  • spawn_position (list[float] | None, optional) – The position to spawn the robot, by default None (origin).

  • small_gap (float, optional) – Small gap to add above the terrain, by default 0.0

  • correct_for_bounding_box (bool, optional) – Whether to adjust the spawn position based on the bounding box, by default True
